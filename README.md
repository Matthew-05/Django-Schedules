
# Django Schedules

An in-development employee manager and scheduler.

## Installation

#### 1. Install postgreSQL
https://www.postgresql.org/download/

#### 2. Install pipenv
```
pip install pipenv --user
```

### 3. CD in and run 
```
pipenv install
```


#### Set Enviroment Variables
```
SECRET_KEY
DATABASE_NAME
DATABASE_USER
DATABASE_PASSWORD
```
